Клиентская библиотека для платформы Tarantool
=============================================

:Версия: |version|

.. sidebar:: Загрузить

    * `PyPI`_
    * `GitHub`_

    **Установить**
    
    .. code-block:: none
    
        $ pip install tarantool


`Tarantool`_ – это очень быстрая платформа in-memory-вычислений.
Изначально разработана в `VK`_ и выпущена под лицензией `BSD`_.
