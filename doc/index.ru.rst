Клиентская библиотека для базы данных Tarantool
===============================================

:Версия: |version|

.. sidebar:: Загрузить

    * `PyPI`_
    * `GitHub`_

    **Установить**
    
    .. code-block:: none
    
        $ pip install tarantool


`Tarantool`_ – это очень быстрая in-memory база данных "ключ-значение".
Изначально разработана в `Mail.Ru`_ и выпущена под лицензией `BSD`_.
