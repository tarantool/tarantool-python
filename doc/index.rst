Python client library for Tarantool Database
============================================

:Version: |version|

.. sidebar:: Download

    * `PyPI`_
    * `GitHub`_

    **Install**

    .. code-block:: none

        $ pip install tarantool


`Tarantool`_ is a damn fast key/value data store originally designed by 
`Mail.Ru`_ and released under the terms of `BSD license`_.
